# AnyImageView
 Đây là 1 bộ thư viện hỗ trợ custommize ImageView trong android   
### Cài đặt  
_file app.grandle_
```groovy 
repositories {
     maven {
        url 'https://dl.bintray.com/dangngocduc/maven'
    }
}

```

thêm trong  _dependencies_
```groovy
compile 'android.dangngocduc:anyimageview:1.0.0'  
```
